# <img src="imgs/appicon.svg" width="36" height="36" alt="icon"> QianPlayer

軽量・高速・多機能な macOS ネイティブ動画プレーヤー。

![メイン画面](imgs/main.png)

## 特徴

### 映像

- **ハードウェアアクセラレーション** — H.264/H.265/VP9/AV1 をハードウェアデコード、CPU 負荷を最小限に
- **HDR 自動適応** — HDR10/HLG コンテンツを検出し、自動トーンマッピングを適用
- **映像パラメータ調整** — 明るさ・コントラスト・彩度・色相・ガンマをリアルタイム調整、グレースケール・回転にも対応
- **字幕システム** — マルチトラック切替、外部 SRT/ASS/SSA/VTT 対応、遅延微調整
- **mpv デコード・Metal ネイティブレンダリング** — mpv コアデコードエンジンと Metal ネイティブレンダーパイプラインによるゼロコピー表示、超低遅延で Apple Silicon GPU の性能を最大限に活用

### 音声

- **マルチトラック切替** — 内蔵されたすべてのオーディオストリームを自由に選択
- **空間オーディオ** — 空間オーディオレンダリングによる没入型サラウンド体験
- **ユニバーサルデコード** — AAC/FLAC/Opus/DTS/AC3/TrueHD など多数対応

### 機能

- **ファイルソース管理** — ローカルフォルダを追加し自動スキャン・インデックス作成、ツリービューで閲覧
- **DLNA / UPnP** — レンダラーとしてスマートフォンや TV ボックスからのキャストを受信
- **マグネットリンク** — マグネットリンクを貼り付けてダウンロードしながら再生、リアルタイムで進捗と速度を表示
- **ピクチャ・イン・ピクチャ (PiP)** — システムレベルのフローティングウィンドウで常に最前面再生
- **プレイリスト** — 作成・管理・並べ替え、再生履歴の自動記録

## システム要件

- macOS 15.0 (Sequoia) 以降
- Apple Silicon (M1 / M2 / M3 / M4)

## インストール

[最新リリース](https://github.com/qianplayer/qianplayer.github.io/releases)をダウンロードし、DMG を開いて QianPlayer を Applications にドラッグしてください。

> 初回起動時にセキュリティ警告が表示された場合：システム設定 → プライバシーとセキュリティ → セキュリティまでスクロール →「このまま開く」をクリックしてください。

## ライセンス

無料・軽量
