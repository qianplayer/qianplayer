# <img src="imgs/appicon.svg" width="36" height="36" alt="icon"> QianPlayer

مشغّل فيديو أصلي لنظام macOS — خفيف وسريع وغني بالميزات.

![الواجهة الرئيسية](imgs/main.png)

## المميزات

### الفيديو

- **فك التشفير بتسريع العتاد** — فك تشفير H.264/H.265/VP9/AV1 عبر العتاد مع استهلاك شبه معدوم للمعالج
- **تكيّف HDR تلقائي** — اكتشاف محتوى HDR10/HLG وتطبيق tone-mapping تلقائي
- **التحكم بمعلمات الفيديو** — ضبط فوري للسطوع والتباين والتشبع ودرجة اللون وغاما؛ مع دعم التدرج الرمادي والتدوير
- **نظام الترجمة** — تبديل متعدد المسارات، دعم ملفات SRT/ASS/SSA/VTT خارجية، ضبط دقيق للتأخير
- **فك تشفير mpv · عرض Metal أصلي** — محرك فك تشفير mpv مع أنبوب عرض Metal أصلي، عرض بدون نسخ، زمن استجابة فائق الانخفاض، واستغلال كامل لوحدة معالجة الرسومات Apple Silicon

### الصوت

- **تبديل متعدد المسارات** — اختيار حر من جميع مسارات الصوت المدمجة
- **الصوت المكاني** — عرض صوت مكاني لتجربة محيطية غامرة
- **فك تشفير شامل** — AAC/FLAC/Opus/DTS/AC3/TrueHD والمزيد

### الوظائف

- **إدارة مصادر الملفات** — إضافة مجلدات محلية مع فحص وفهرسة تلقائية، تصفح شجري
- **DLNA / UPnP** — يعمل كمستقبِل لبث الشاشة من الهواتف وأجهزة التلفزيون
- **روابط Magnet** — الصق رابط Magnet للتشغيل أثناء التحميل مع عرض فوري للتقدم والسرعة
- **صورة داخل صورة (PiP)** — نافذة عائمة على مستوى النظام تبقى دائمًا في المقدمة
- **قوائم التشغيل** — إنشاء وإدارة وترتيب القوائم مع سجل تشغيل تلقائي

## متطلبات النظام

- macOS 15.0 (Sequoia) أو أحدث
- Apple Silicon (M1 / M2 / M3 / M4)

## التثبيت

حمّل [أحدث إصدار](https://github.com/qianplayer/qianplayer.github.io/releases)، افتح ملف DMG واسحب QianPlayer إلى مجلد التطبيقات.

> إذا منع macOS التشغيل لأول مرة: افتح إعدادات النظام ← الخصوصية والأمان ← مرّر للأسفل حتى قسم الأمان ← انقر على «فتح على أي حال» لـ QianPlayer.
