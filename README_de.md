# <img src="imgs/appicon.svg" width="36" height="36" alt="icon"> QianPlayer

Ein leichtgewichtiger, schneller und funktionsreicher nativer Video-Player für macOS.

![Hauptoberfläche](imgs/main.png)

## Funktionen

### Video

- **Hardwarebeschleunigte Dekodierung** — Hardware-Dekodierung von H.264/H.265/VP9/AV1 bei minimaler CPU-Auslastung
- **Automatische HDR-Anpassung** — Erkennung von HDR10/HLG-Inhalten mit automatischem Tone-Mapping
- **Video-Parametersteuerung** — Echtzeit-Anpassung von Helligkeit, Kontrast, Sättigung, Farbton und Gamma; Graustufen- und Rotationsmodus
- **Untertitelsystem** — Mehrspurumschaltung, externe SRT/ASS/SSA/VTT-Unterstützung, Feineinstellung der Verzögerung
- **mpv-Dekodierung · Natives Metal-Rendering** — Basierend auf dem mpv-Dekodierungskern mit nativer Metal-Render-Pipeline, Zero-Copy-Anzeige, ultraniedrige Latenz und volle Nutzung der Apple Silicon GPU

### Audio

- **Mehrspurauswahl** — Freie Auswahl aller eingebetteten Audiostreams
- **Spatial Audio** — Räumliches Audio-Rendering für ein immersives Surround-Sound-Erlebnis
- **Universelle Formatdekodierung** — AAC/FLAC/Opus/DTS/AC3/TrueHD und mehr

### Funktionalität

- **Dateiquellenverwaltung** — Lokale Ordner hinzufügen mit automatischem Scannen und Indexieren, Baumansicht-Navigation
- **DLNA / UPnP** — Fungiert als Renderer zum Empfang von Bildschirmübertragungen von Smartphones und TV-Boxen
- **Magnet-Links** — Magnet-Link einfügen, um während des Downloads zu streamen, mit Echtzeit-Fortschritt und Geschwindigkeitsanzeige
- **Bild-in-Bild (PiP)** — Schwebendes Systemfenster für Wiedergabe im Vordergrund
- **Wiedergabelisten** — Erstellen, Verwalten und Sortieren von Wiedergabelisten mit automatischem Verlauf

## Systemanforderungen

- macOS 15.0 (Sequoia) oder neuer
- Apple Silicon (M1 / M2 / M3 / M4)

## Installation

Laden Sie die [neueste Version](https://github.com/qianplayer/qianplayer.github.io/releases) herunter, öffnen Sie das DMG und ziehen Sie QianPlayer in den Ordner „Programme".

> Falls macOS den ersten Start blockiert: Öffnen Sie Systemeinstellungen → Datenschutz & Sicherheit → scrollen Sie zu Sicherheit → klicken Sie auf „Trotzdem öffnen" für QianPlayer.
