# <img src="imgs/appicon.svg" width="36" height="36" alt="icon"> QianPlayer

輕量、快速、功能豐富的 macOS 原生影片播放器。

![主介面](imgs/main.png)

## 特色

### 視訊

- **硬體加速解碼** — 硬解 H.264/H.265/VP9/AV1，CPU 佔用極低
- **HDR 自動適配** — 識別 HDR10/HLG 內容，自動 tone-mapping
- **視訊參數調節** — 亮度、對比度、飽和度、色相、伽瑪即時調整，支援灰階與旋轉
- **字幕系統** — 多軌切換，外掛 SRT/ASS/SSA/VTT，延遲微調
- **mpv 解碼 · Metal 原生呈現** — 基於 mpv 核心解碼引擎，Metal 原生渲染管線零拷貝上屏，延遲極低，充分發揮 Apple Silicon GPU 效能

### 音訊

- **多音軌切換** — 自由選擇影片內嵌的所有音訊串流
- **空間音訊** — 支援空間音訊渲染，沉浸式環繞聲體驗
- **全格式解碼** — AAC/FLAC/Opus/DTS/AC3/TrueHD 等

### 功能

- **檔案來源管理** — 新增本機資料夾，自動掃描索引，樹狀瀏覽
- **DLNA / UPnP** — 作為渲染器接收投放，手機/電視盒直推播放
- **磁力連結** — 貼上磁力連結邊下邊播，即時進度/速率顯示
- **子母畫面 (PiP)** — 系統級浮窗置頂播放
- **播放清單** — 建立、管理、排序，自動記錄播放歷史

## 系統需求

- macOS 15.0 (Sequoia) 或更高版本
- Apple Silicon (M1 / M2 / M3 / M4)

## 安裝

下載[最新版本](https://github.com/qianplayer/qianplayer.github.io/releases)，開啟 DMG 拖入 Applications 即可。

> 首次開啟如遇安全性提示：開啟「系統設定」→「隱私權與安全性」→ 下滑到「安全性」部分 → 點選「仍要打開」QianPlayer。
